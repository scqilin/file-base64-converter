图片转换工具

支持JPG、PNG、GIF等常见图片格式与Base64之间的相互转换

🖼️ 图片 → Base64

🖼️

请选择图片文件

使用说明：
1. 点击"上传图片"选择文件，或点击"粘贴图片"从剪贴板获取
2. 支持 JPG、PNG、GIF、BMP、WebP 等常见格式
3. 自动转换为Base64格式，并保留原始文件名信息
4. 转换的Base64包含文件名，还原时会自动使用原始文件名
5. 所有处理都在本地完成，确保文件安全

🔄 Base64 → 图片

🖼️

图片预览区域

使用说明：
1. 粘贴包含完整data:image前缀的Base64编码
2. 如果Base64包含文件名信息，会自动识别并显示原始文件名
3. 点击"转换为图片"生成预览
4. 确认无误后点击"下载图片"保存到本地（使用原始文件名）
5. 支持自动识别图片格式并设置正确的文件扩展名

📦 压缩文件 → Base64

📦

请选择压缩文件

使用说明：
1. 支持常见压缩格式：ZIP、RAR、7Z、TAR、GZ、BZ2、XZ
2. 上传后自动转换为Base64，并保留原始文件名
3. 转换的Base64包含文件名信息，还原时自动使用原名
4. 所有处理都在浏览器本地完成，不会上传到服务器

🔄 Base64 → 压缩文件

📦

文件预览区域

使用说明：
1. 粘贴压缩文件的Base64编码（支持自动识别文件名）
2. 点击"转换为文件"解析Base64数据
3. 如果包含文件名信息，会自动使用原始文件名
4. 点击"下载文件"保存到本地，保持原始格式
